Blocos são definidos por {}

O que define uma variável global, local, e sua tipagem é a foma como você "declara ela"

var = global
let = local
const = constante (imutável)(tem sua função melhor utilizada no react)


-> Tipos Primitivos:

  -> numérico: números
  -> string: cadeira de caracteres
  -> boolean: true or false
  -> qualquer: quando você não especifica o tipo da variável


Exemplos explicativos:

var hihi = true; (tipo booleano) //ou false
var hehe = 'hehheeheh'; (string)
const i = 10; (constante)
var ihihihi; (qualquer)
var hehhehehe = 10; (numerico)
